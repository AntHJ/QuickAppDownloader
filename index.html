Manual Update Instructions

  1. Open the Settings app

  2. Select "System Update" and then "Update Using Wi-Fi."

  3. You should see a pop-up saying there is a new update. Make sure the new update version is "3.74"! If the version is different, do not proceed.

  4. After ensuring you are updating to 3.74, proceed with the steps on screen to update.

  5. After updating to 3.74, you can revisit this website to jailbreak your console.
